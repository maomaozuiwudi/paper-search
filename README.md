# 论文秒搜 PaperSearch

多平台学术论文搜索工具，8大平台并发搜索，FTS5本地缓存，一键直达原文。

## 支持平台

- OpenAlex - 开放学术图谱
- Semantic Scholar - 语义学者
- arXiv - 预印本
- PubMed - 生物医学
- DOAJ - 开放获取期刊
- 百度学术 - 中文期刊
- CiNii - 日本学术
- CORE - 聚合仓储

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```
